## 介绍

基于 [webpack5](https://github.com/webpack/webpack) 和 [typeScript5](https://github.com/microsoft/typeScript) 和 [jest29](https://github.com/jestjs/jest) 搭建

## 特性

- 开发环境
- 生产环境
- 测试环境
- 自定义环境变量
- 打包分析
- 生产环境低版本浏览器支持（当前项目内构建的产物可运行在ie11）

## 外部自定义环境变量

- **.env** 基础环境
- **.env.development** 开发环境
- **.env.production** 生产环境
